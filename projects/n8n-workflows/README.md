## ğŸš€ Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/HamzaGujjar579/n8n-workflows.git
   cd n8n-workflows
   ```

2. **Install n8n**
   ```bash
   npm install n8n -g
   ```

3. **Import workflows**
   ```bash
   # Import individual workflow files
   n8n import:workflow --input=workflows/crm-sync.json
   ```

4. **Start n8n**
   ```bash
   n8n start
   ```
