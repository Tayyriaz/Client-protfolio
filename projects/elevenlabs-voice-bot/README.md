## üöÄ Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/HamzaGujjar579/elevenlabs-voice-bot.git
   cd elevenlabs-voice-bot
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure API keys**
   ```bash
   cp .env.example .env
   # Add your ElevenLabs and OpenAI API keys
   ```

4. **Run the bot**
   ```bash
   python main.py
   ```
