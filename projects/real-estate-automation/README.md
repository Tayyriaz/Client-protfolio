1. **Clone the repository**
   ```bash
   git clone https://github.com/HamzaGujjar579/real-estate-automation.git
   ```

   ```bash
   pip install -r requirements.txt

   ```

   ```bash

   ```

4. **Run the application**
   ```bash
   python manage.py runserver
   npm start
   ```
